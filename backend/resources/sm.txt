左侧拖拽节点到画布可创建节点
画布空白处右键 → 快速添加节点
从节点输出口拖到输入口可创建连线
连线类型不符或输入口已占满将被拒绝
图片输出 → 视频生成节点（绿→紫）
视频输出 → 视频输出节点（蓝→灰）
图片可拖拽到 Gemini 橙色输入口（最多14张）
拖拽缩略图可改变参考图顺序
可从文件管理器拖拽图片/视频/音频到画布
拖入多张文件会创建多个上传节点
图片节点右键 → 复制图片 / 绘制蒙版 / 打开文件夹
视频/输出节点右键 → 用系统播放器打开 / 打开文件夹
输出节点右键 → 复制路径（带 TAPNOW_IMG/VID 前缀）
画布粘贴 TAPNOW 路径会创建对应的图片/视频上传节点
单击视频缩略图节点内播放，再次点击暂停
双击视频缩略图用系统播放器打开
拖拽视频缩略图到资源管理器/微信可发送
输出/文本节点右键执行当前分支或工作流
多选输出节点后工具栏「执行」并行执行
工具栏「队列」查看任务进度，双击行定位节点
工具栏「历史」查看历史，双击加载参数
模板可点击画布任意位置放置（不清空现有内容）
Ctrl+Z/Y 撤销/重做；Ctrl+S 立即保存
Ctrl+C/V 复制粘贴节点（含内部连线）
Delete 删除选中节点和连线
Ctrl+A 全选；Ctrl+点击全选连接的所有节点
方向键微调节点位置（10px）
鼠标滚轮缩放，鼠标中键/右键拖拽平移画布
执行中节点边框流光，输出节点头部蓝/绿/红
图片自动压缩 ≤4.7MB 后再发送给 API
API 设置 → 通用设置可关闭右侧说明面板
